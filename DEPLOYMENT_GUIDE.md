1. Go to: **https://supabase.com**
2. Click **"Start your project"**
3. Click **"Sign in with GitHub"** (use your GitHub account)
4. Click **"New Project"**
5. Fill in:
   - Organization: Click "New organization" → Name it: `IT Academies`
   - Project name: `it-academies-nam`
   - Database Password: **CREATE A STRONG PASSWORD AND SAVE IT!**
   - Region: Choose **"South Africa (Cape Town)"** (closest to Namibia)

7. **WAIT 2-3 MINUTES** (database is being created)

### **STEP 4: SETUP DATABASE TABLES (5 minutes)**

1. In Supabase, click **"SQL Editor"** (left sidebar)
2. Click **"New query"**
3. Go to your GitHub repo: **https://github.com/IT-ACADEMIES/IT-ACADEMIES**
4. Open the file: **`database/schema.sql`**
5. Click **"Raw"** button
6. **Copy ALL the text**
7. Go back to Supabase SQL Editor
8. **Paste it** into the editor
9. Click **"Run"** (or press Ctrl+Enter)
10. You should see: **"Success. No rows returned"**
11. ✅ Done! Database is ready
